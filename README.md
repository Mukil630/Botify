## ✨ Features

- 🤖 **AI-powered replies** using Groq LLM — context-aware responses based on business data
- 📅 **WhatsApp booking system** — customers book appointments directly via chat
- ⚡ **Instant QR connection** — no Meta API approval, bot live in seconds
- 💳 **Tiered SaaS plans** — Free / Starter / Pro / Business with daily limits
- 📊 **Real-time message tracking** — live progress bar with daily quota
- 👑 **Admin dashboard** — manage users, approve payments, track revenue
- 🔒 **Multi-tenant architecture** — complete data isolation per business
- 🛎️ **Service menu** — bot auto-shows business services to customers
- 🏢 **Business info** — address, timings fed to AI for smart replies

## 🛠️ Tech Stack

| Layer      | Technology                          |
|------------|-------------------------------------|
| Backend    | Python, Flask 3.1                   |
| Database   | PostgreSQL + SQLAlchemy 2.0         |
| AI         | Groq LLM API                        |
| Bot Engine | Node.js + whatsapp-web.js           |
| Auth       | Flask-Login + Session management    |
| Frontend   | Jinja2, HTML5, CSS3, Vanilla JS     |
| Deploy     | Railway (Production)                |
